About

I've had some other people ask me if I knew a place to find multicraft jar configs. I didn't, and haven't been able to find one. So this week, I asked again by Luke over at PPX Hosting. I decided, since I've been wanting to learn Python for a while and this site would be simple enough, to make one.

So, I present to you http://multicraft.ix.gd/! This simple site is both a generator and a sharing platform for Multicraft's .jar.conf files. It's also open source, on Github. I had fun making it, and I hope you server admins out there find it useful.

This site was built by Connor Peet and is hosted by FerrousMCM
